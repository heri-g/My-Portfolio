NgSwitch

ngSwitch is comprised of two directives, an attribute directive and a structural directive. Here we see the ngSwitch attribute directive being attached to an element. This expression bound to the directive defines what will compared against in the switch structural directives. If an expression bound to ngSwitchCase matches the one given to ngSwitch, those components are created and the others destroyed. If none of the cases match, then components that have ngSwitchDefault bound to them will be created and the others destroyed.
